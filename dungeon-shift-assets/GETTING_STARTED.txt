🎯 USAGE TIPS
═══════════════════════════════════════════════════════════════════════════════

✓ Use integer scaling (2×, 3×, 4×) for crisp pixels
✓ Disable texture filtering in your game engine
✓ Animate walk cycles at 8-12 FPS
✓ Animate idle breathing at 4-6 FPS
✓ Layer order: Floor → Objects → Entities → Walls → UI
✓ Use smaller collision boxes than sprite size


🎨 CUSTOMIZATION
═══════════════════════════════════════════════════════════════════════════════

Want different colors or designs?

1. Install Python 3.6+ and Pillow:
   pip install Pillow

2. Edit any generate_*.py script:
   - Change colors in PALETTE dictionary
   - Modify draw_* functions for different shapes

3. Regenerate assets:
   python generate_all.py

4. See your changes instantly!
